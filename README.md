# TechNova Password Generator

## Project Overview
This project is a Python Password Generator developed as part of the TechNova Python Programming Internship.

## Features
- User chooses password length
- Generates strong and secure passwords
- Uses uppercase letters, lowercase letters, numbers, and special characters
- Saves generated passwords to a text file

## Technologies Used
- Python 3
- random
- string

## How to Run

1. Run the program:
   ```bash
   python password_generator.py
   ```
2. Enter the desired password length.
3. The generated password will be displayed.
4. The password will also be saved in `passwords.txt`.
